Display a different theme to the user if the user is logged in as an admin.

== Description ==

“Theme Demo for Admins” is a new WordPress plugin created by Revive Media Services and available for free for download. It allows you to specify a different theme for users logged in as an admin without affecting the view of your regular users.

== Installation ==

1. Upload `theme_demo.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the 'Theme Demo' admin settings page and choose your theme for admins browsing he site

== Frequently Asked Questions ==

= What is an example usage of this plugin? =

We recently had a client request a complete redesign of every page of their WordPress site. Using this plugin, the client is able to see their new website in its entirety using their new look while the public customers to their website still see the look they are familiar with.
